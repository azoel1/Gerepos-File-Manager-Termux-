# Gerepost FM 📱💼

Легковесный, интерактивный консольный (CLI) файловый менеджер, созданный специально для удобной работы в терминалах на Android и ограниченных средах.

## 🚀 Особенности
* **Универсальное управление кнопками:** Скрипт одновременно понимает как стандартные стрелочки, так и буквенную раскладку WASD (удобно для ввода с обычной мобильной клавиатуры, где нет стрелок).
* **Умная фильтрация:** Автоматически разделяет типы файлов красивыми эмодзи (📁 для папок, 📄 для файлов).
* **Абсолютная легковесность:** Работает мгновенно, запускается напрямую через Node.js и не требует установки тяжелых зависимостей.

## 🎮 Управление (Раскладка)

| Действие | Стандартные клавиши | Альтернатива (Для мобильной клавы) |
| --- | --- | --- |
| Листать вверх 🔼 | Стрелка Вверх (↑) | Клавиша W / w |
| Листать вниз 🔽 | Стрелка Вниз (↓) | Клавиша S / s |
| Открыть / Войти ➡️ | Клавиша Enter | Клавиша D / d |
| Назад (Вверх по папке) ⬅️ | Выбор пункта .. | Клавиша A / a |
| Выйти из менеджера ❌ | Комбинация Ctrl + C | Клавиша Q / q |

## 🛠️ Запуск на Android (В Termux)

Чтобы запустить менеджер прямо на телефоне, выполните следующие команды:

1. Установите Node.js (если еще не установлен):
'pkg install nodejs'

2. Запустите скрипт из папки проекта:
'node server.js'
