# Gerepost FM 📱💼

Легковесный, интерактивный консольный (CLI) файловый менеджер, созданный специально для удобной работы в терминалах на Android и ограниченных средах.

## 🚀 Особенности
* **Универсальное управление:** Скрипт полностью управляется как стандартными стрелочками клавиатуры, так и привычной раскладкой WASD. Это делает его максимально удобным для ввода с обычных мобильных клавиатур на смартфонах, где физических стрелок просто нет.
* **Умная фильтрация:** Автоматически разделяет типы файлов красивыми эмодзи (📁 для папок, 📄 для файлов).
* **Абсолютная легковесность:** Работает мгновенно, запускается напрямую через Node.js и не требует установки тяжелых зависимостей.

## 🎮 Управление (Раскладка)

| Действие | Клавиши управления |
| --- | --- |
| Листать вверх 🔼 | Стрелка Вверх (↑) / Клавиша W или w |
| Листать вниз 🔽 | Стрелка Вниз (↓) / Клавиша S или s |
| Открыть / Войти ➡️ | Клавиша Enter / Клавиша D или d |
| Назад (Вверх по папке) ⬅️ | Выбор пункта .. / Клавиша A или a |
| Выйти из менеджера ❌ | Комбинация Ctrl + C / Клавиша Q или q |

## 🛠️ Запуск на Android (В Termux)

Чтобы запустить менеджер прямо на телефоне, выполните следующие команды:

1. Установите Node.js (если еще не установлен):
'pkg install nodejs'

2. Запустите скрипт из папки проекта:
'node server.js'
